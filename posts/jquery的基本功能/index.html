jQuery的基本功能

1. jQuery 如何获取元素

jQuery的基本设计思想和主要用法，就是获取某个网页元素，然后对其进行某种操作.

  • 使用jQuery,一般就是将一个选择表达式，放进构造函数jQuery()（简写为$），然后得到被选中的元素。选择表达式可以是CSS选择器,也可以是jQuery特有的表达式
//css表达式
$(document) //选择整个文档对象
$('#myId') //选择ID为myId的网页元素
$('div.myClass') // 选择class为myClass的div元素
$('input[name=first]') // 选择name属性等于first的input元素

//jQuery特有的表达式
$('a:first') //选择网页中第一个a元素
$('tr:odd') //选择表格的奇数行
$('#myForm :input') // 选择表单中的input元素
$('div:visible') //选择可见的div元素
$('div:gt(2)') // 选择所有的div元素，除了前三个
$('div:animated') // 选择当前处于动画状态的div元素

2. jQuery的链式操作

链式操作就是最终选中网页元素以后，可以对它进行一系列操作，并且所有操作可以连接在一起，以链条的形式写出来.

  • $('#xxx’)返回的值,并不是元素,是一个api对象
//先找到div元素, 选择其中的h3元素, 选择第3个h3元素, 将这个h3元素的内容改为Hello.
$('div').find('h3').eq(2).html('Hello');


const api1 = $(".test");
const api2 = api1.find(".child").addClass("red");
//可写成这样
$(".test").find(".child").addClass("red");
  • jQuery还提供了.end()方法, 使获取的元素可以返回上级
//退回到选中所有的p元素的那一步
$('div')
　　　.find('p')
　　　.eq(1)
　　　.html('Hello')
　　　.end()

3. jQuery如何创建元素

jQuery创建元素, 把新元素直接穿入jQuery的构造函数中

//创建一个div包含span的元素
$("<div><span>1</span></div>")
//创建一个class为new的li元素
$('<li class="new">new list item</li>')
//将li元素添加为ul元素的子元素
$('ul').append('<li>list item</li>');

4. jQuery如何移动元素

jQuery在移动元素上提供了两种方法,一种是直接移动该元素到指定位置, 二种是移动其他元素,让目标元素到达目的地. 假定我们选中了一个div元素，需要把它移动到p元素后面。

  • 1.使用.insertAfter()，把div元素移动p元素后面：
$('div').insertAfter($('p'));
  • 2..after()，把p元素添加到div元素前面
$('p').after($('div'));

两者的区别是.insertAfter()返回div元素，.after()返回p元素. 使用这种模式的操作方法，一共有四对：

.insertAfter()和.after()：    在现存元素的外部，从后面插入元素
.insertBefore()和.before()：  在现存元素的外部，从前面插入元素
.appendTo()和.append()：      在现存元素的内部，从后面插入元素
.prependTo()和.prepend()：    在现存元素的内部，从前面插入元素

5. jQuery 如何修改元素的属性

  • 1.text() 设置或返回所选元素的文本内容
  • 2.html() 设置或返回所选元素的内容（包括 HTML 标记)
  • 3.val() 设置或返回表单字段的值
  • 4.width() 取出或设置某个元素的宽度
  • 5.height() 取出或设置某个元素的高度

以上5个函数说明很清楚了 赋值(括号内不为空为赋值）或者取值（括号内为空为取值). 如果结果集包含多个元素，那么赋值的时候，将对其中所有的元素赋值；取值的时候，则是只取出第一个元素的值（.text()例外，它取出所有元素的text内容）。

  • 6.attr() 方法用于获取或修改属性值，但是跟上面三个函数有所不同
//取值
$("#test").attr("href")
//设值
$("#test").attr("href","https://www.baidu.com")
